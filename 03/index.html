Властивість font-size

Властивість font-size змінює розмір шрифта.

Розмір шрифту задається за допомогою значення та одиниць виміру (em, px, pt, %)

Синтаксис

CSS

p{
 font-size: 20px;
 }

HTML

<p>Hello, world!</p>

Результат

Hello, world!

Властивість font-weight

Властивість font-weight встановлює насиченість шрифту.

Може приймати наступні значення:

  • normal - нормальна насиченість, те саме, що і 400
  • bold - напівжирний текст, те саме, що і 700
  • lighter - змінює насиченість відносно батьківського стилю в меншу сторону
  • bolder - змінює насиченість відносно батьківського стилю в більшу сторону
  • 100, 200, 300, 400, 500, 600, 700, 800, 900 - цифрові значення насиченості, які відрізняються від стандартних

Синтаксис

CSS

.bold{
 font-weight: bold;
 }
 .thin{
 font-weight: 100;
 }
 .max{
 font-weight: 900;
 }

HTML

<p>
 <span class = "bold">Напівжирний текст</span>
 <span class = "thin">найтонший текст</span>
 <span class = "max">найтовщий текст</span>
 </p>

Результат

Напівжирний текст, найтонший текст, найтовщий текст

Властивість font-style

Властивість font-style визначає накреслення шрифту - звичайне, курсивне або похиле.

Може приймати наступні значення:

  • normal - звичайне зображення тексту
  • italic - курсивне зображення
  • oblique - похиле накреслення. Курсив і похилий шрифт при всій їхній схожості не одне й те саме. Курсив це спеціальний шрифт, що імітує рукописний, а похилий утворюється шляхом нахилу звичайних знаків вправо

Синтаксис

CSS

.italic{
 font-style: italic;
 }

HTML

<span class = "italic">Текст курсивом</span>

Результат

Текст курсивом

Властивість font-family

Властивість font-family встановлює сімейство шрифту, яке використовуватиметься для оформлення текстового вмісту. Список шрифтів може містити одну або кілька назв, розділених комою.

Може приймати наступні значення:

  • serif - шрифти із засічками (антиквенні), типу Times;
  • sans-serif - рубані шрифти (шрифти без засічок чи гротески), типовий представник Arial;
  • cursive - курсивні шрифти;
  • fantasy - декоративні шрифти;
  • monospace - моноширинні шрифти, ширина кожного символу у такому сімействі однакова (шрифт Courier).

Синтаксис

CSS

.serif{
 font-family: serif;
 }

HTML

<span class = "serif">Шрифт типу Times</span>

Результат

Шрифт типу Times

Властивість color

Властивість color визначає колір тексту елемента.

Синтаксис

CSS

div{
 color: #29438e;
 }

HTML

<div>Колір тексту</div>

Результат

Колір тексту

Властивість text-transform

Властивість text-transform керує перетворенням тексту елемента у великі або великі символи. Коли значення відрізняється від none, регістр вихідного тексту буде змінено.

Може приймати наступні значення:

  • capitalize - перший символ кожного слова у реченні буде великим. Інші символи свого вигляду не змінюють.
  • lowercase - всі символи тексту стають малими (нижній регістр).
  • uppercase - всі символи тексту стають великими (верхній регістр).
  • none - не змінює регістр символів.

Синтаксис

CSS

div{
 text-transform: none;
 }

HTML

<div>Текст великими або маленькими літерами</div>

Результат

Текст великими або маленькими літерами

Властивість text-decoration

Властивість text-decoration додає оформлення тексту як його підкреслення, перекреслення чи лінії над текстом. Одночасно можна застосувати більше одного стилю, перераховуючи значення через пропуск.

Може приймати наступні значення:

  • line-through - створює перекреслений текст.
  • overline - лінія проходить над текстом.
  • underline - встановлює підкреслений текст.
  • none - скасовує всі ефекти, у тому числі й підкреслення у посилань, яке задано за замовчуванням.

Синтаксис

CSS

div{
 text-decoration: none;
 }

HTML

<div>Текст</div>

Результат

Перекреслений текст
